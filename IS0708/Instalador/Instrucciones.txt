Para poder ejecutar el instalador, si tienes instalado Java 6,  tiene que estar
en el mismo directorio el instalador de java "jre-6u5-windows-i586-p-s.exe".

Para compilarlo, hay que tener instalado el compilador de NSIS y su plugin para
el eclipse.
    - NSIS: http://nsis.sourceforge.net/Download
    - Plugin para eclipse: http://eclipsensis.sourceforge.net/index.shtml
