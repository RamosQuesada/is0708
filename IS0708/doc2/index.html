Descripción

TurnoMatic es una base de datos departamental y un gestor de automatización de horarios.

La aplicación soluciona los clásicos problemas que surgen a la hora de realizar el cuadrante de empleados de una empresa en la que los empleados no tienen un horario fijo y estos trabajan por horas. También permite la gestión de empleados, visualización del rendimiento tanto de los empleados como de los departamentos y además, ofrece un servicio de mensajería interna.

El programa está diseñado para ser usado por cualquier empleado en la jerarquía de una empresa, desde el gerente hasta el empleado de menor rango. El gerente supervisa, con este programa, cada departamento de su empresa, tanto su rendimiento como la distribución de personal. Los jefes de departamento pueden tener controlado en todo momento el horario de sus empleados, pudiendo modificarlo a su gusto y crear informes de rendimiento. Los empleados de la empresa pueden ver en todo momento sus horarios, realizar peticiones de cambio de horario a través del sistema y consultar sus registros de ventas.

Puntos a destacar

  • Algoritmo de generación de horarios con backtracking que contempla no sólo la correción de los horarios generados, sino la comodidad de los empleados en la medida de lo posible.
  • Base de datos externa, compatible con MySQL. La construcción modular facilita la migración a otro sistema de base de datos.
  • Interfaz propio construido desde cero. Intuitivo y adaptado exactamente a las necesidades específicas de los clientes.
  • Sistema opcional de impresión y lectura de códigos de barras para agilizar el uso de la aplicación y garantizar la seguridad al no tener que teclear contraseñas.
  • Interfaz web para consultas remotas.
  • El instalador para Windows desarrollado con NullSoft Scriptable Install System permite una cómoda y rápida instalación de la aplicación y la JVM si fuera necesaria.

Compatibilidad

  • Desarrollado en Java 1.6.
  • Compatible con Windows Me, XP, 2000 y Vista, Linux 32 y 64 bits, Solaris 8, Mac OS X y AIX.
  • Compatible con MySQL y PHP 5.5.
  • Construcción modular que facilita la migración a otro sistema de base de datos.
  • Paquetes de idiomas independientes para facilitar la traducción.
  • El interfaz, desarrollado con el Standard Widget Toolkit, se integra perfectamente con cualquier sistema operativo, resultando en un aspecto familiar al usuario.
