Tecnologías

La aplicación está desarrollada usando el lenguaje de programación Java mediante la plataforma Eclipse. Utiliza la librería SWT para los interfaces, lo que permite que la aplicación se adapte visualmente al entorno del sistema operativo, independientemente de la plataforma.

  • Página oficial de Java: http://java.sun.com/
  • Página oficial de Eclipse: http://www.eclipse.org/
  • La base de datos estará gestionada mediante MySQL.
  • Página oficial de MySQL: http://www.mysql.com/
  • El entorno Web está programado en HTML y PHP.
  • Página oficial de PHP: http://www.php.net/
  • El instalador ha sido realizado con Nullsoft Scriptable Install System (NSIS)
  • Página oficial de NSIS: http://nsis.sourceforge.net/
  • La identificación del usuario se hará mediante un lector óptico de códigos de barras.
