Inicio : Configuración de la base de datos

Es la pantalla de inicio del menú de administrador. En ella se encuentran opciones para configurar la base de datos. Estas opciones son:

  • Configurar la base de datos de la aplicación.
  • Resetear la base de datos.

Configurar la base de datos de la aplicación:

Esta opción permite elegir la base de datos a usar en la aplicación suponiendo que se puedan usar varias bases de datos situadas en distintas localizaciones (por ejemplo la base de datos se encuentra en un servidor pero las pruebas pueden hacerse sobre el localhost).

Al pulsar el botón "Config BD" se abre una nueva ventana donde podremos configurar los parámetros de la base de datos a usar en la aplicación.  Los parámetros se guardan en un archivo de texto que la aplicación lee cada vez que se inicia, por tanto la última base de datos asignada será la que se cargue en la siguiente ejecución de la aplicación.

Los parámetros a configurar son los siguientes:

  • Server IP : Introducir la IP del servidor donde se encuentra la base de datos. En el caso de usar el localhost usar "localhost".
  • Server Username : El nombre de usuario para acceder a la base de datos. Si se está usando el localhost poner "root".
  • Server Password: Contraseña del servidor. Se utiliza un algoritmo de encriptado para mayor seguridad.
  • Admin Password: Si el administrador desea cambiar la contraseña lo puede hacer aquí. Se utiliza un algoritmo de encriptado para mayor seguridad.

Los cambios surtirán efecto en la siguiente ejecución de la aplicación por lo que si se necesita usar la nueva base de datos de inmediato hay que salir y volver a entrar.

Resetear la base de datos:

Esta opción permite resetear la base de datos. Se usará cuando es necesario actualizar información en grandes cantidades o para pruebas (existe un programa de prueba que rellena la base de datos con información aleatoria y otro programa que la rellena con una información predeterminada para pruebas).
