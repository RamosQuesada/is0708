Creación de un nuevo jefe

La pestaña "admin: Nuevo Jefe"  le permitirá configurar un nuevo jefe para insertarlo en la Base de datos. Para ello deberá rellenar todos los campos que se le muestran en esa pestaña.

Para confirmar la creación del empleado pulse el botón "aceptar", en caso contrario pulse "cancelar" para borrar todos los datos configurados. En caso de no haber rellenado todos los campos de la pestaña o de haberlo hecho de forma incorrecta, al pulsar el botón "aceptar" aparecerá una ventana indicando que campo esta aún sin rellenar o que errores tiene.

A continuación se detallará el funcionamiento y restricciones de los diferentes elementos.

En la pantalla podrá diferenciar dos recuadros para configurar dos tipos de datos

Datos personales:   Permite configurar los datos personales del usuario, no relacionados con su contrato o su puesto laboral.

  • Vendedor:
  • Clave:
  • Nombre, 1er apellido, 2º apellido:
  • fecha de nacimiento:
  • sexo:
  • idioma:

Datos Laborales: Permite configurar los datos relacionados con el contrato y su puesto laboral.

  • Tipo de contrato:
  • Experiencia:
  • Fecha inicial de contrato:
  • Fecha de alta:
  • Color:

 

Si tiene alguna duda sobre como seleccionar una fecha pulse aquí.

 

Vendedor:

Se corresponde con el número de vendedor asignado al usuario, este número deberá ser una cifra de 8 dígitos y será el número con el cual el usuario accederá a la aplicación para loguearse.

 

 

volver al principio.

Clave:

Este campo corresponde a la clave personal de un jefe, la cual utilizara junto con su número de vendedor para acceder tanto a la aplicación como a la página web. Esta clave no podrá tener una longitud mayor de 10 caracteres, estos caracteres podrán ser tanto letras como números .

para rellenar este campo se admiten dos opciones:

  • Generación automática: la aplicación le sugerirá una clave de 8 caracteres
  • Generación manual: la aplicación le permitirá introducir por teclado la clave que usted desee.

 

 

volver al principio.

Nombre, 1er apellido, 2º Apellido :

Estos campos se corresponderán con el nombre, 1er apellido, 2º Apellido del usuario que estamos introduciendo, estos campos permiten una longitud de 20 caracteres cada uno de ellos.

 

volver al principio.

fecha de nacimiento:

Para seleccionar la fecha de nacimiento del empleado pulse sobre el botón "Fecha de nacimiento" del recuadro datos personales, al pulsar este botón aparecerá una ventana que le permitirá seleccionar la fecha de nacimiento correspondiente.

Si necesita mas información sobre como introducir la fecha pulse aquí.

 

volver al principio.

Sexo:

Esta opción tiene un menú desplegable que le permitirá configurar el sexo del empleado, por defecto aparecerá la opción de sexo masculino.

 

volver al principio.

Idioma:

Esta opción le permitirá configurar el idioma con el cual el jefe que está creando accederá a la aplicación.

 

volver al principio.

Tipo de contrato:

Esta opción le permitirá seleccionar un contrato de la lista de contratos disponibles para configurar el contrato del jefe que esta creando, si desea crear un nuevo contrato cambie a la pestaña de contratos, la cual le permitirá crear, modificar y eliminar un contrato.

 

volver al principio.

Experiencia:

aun por determinar

volver al principio.

Fecha inicial de contrato:

Para seleccionar la fecha de inicio del contrato seleccionado en "Tipo de contrato" del empleado que está creando pulse sobre el botón "Seleccionar fecha inicial de contrato" del recuadro datos laborales, al pulsar este botón aparecerá una ventana que le permitirá seleccionar la fecha correspondiente.

Si necesita mas información sobre como introducir la fecha pulse aquí.

 

volver al principio.

Fecha de alta:

La fecha de alta se corresponderá con la fecha de inicio de contrato, en caso de ser la primera vez que se de de alta a un usuario en la aplicación.

Para seleccionar la fecha de alta del usuario en la empresa pulse sobre el botón "Seleccionar fecha de alta" del recuadro datos laborales, al pulsar este botón aparecerá una ventana que le permitirá seleccionar la fecha correspondiente.

Si necesita mas información sobre como introducir la fecha pulse aquí.

 

volver al principio.

 

Fechas:

Al haber pulsado una de las siguientes opciones:

  • Fecha de nacimiento
  • Seleccionar fecha de inicio de contrato
  • Seleccionar fecha de alta

o cualquier otra que despliegue la ventana del calendario, podrá cambiar de fecha de la siguiente forma.

  • Año: para cambiar de año pulse con el botón derecho del ratón sobre el año que se muestra en la ventana, el cual le permitirá ver unas flechas que le permitirán desplazarse hacia arriba (avanzar de año)  o hacia abajo ( retroceder año) en los años, en caso de dejar pulsado una de las dos flechas avanzar de forma más rápida para llegar al año de la fecha deseada.
  • Mes: para cambiar de mes pulse con el botón derecho del ratón sobre el mes que se muestra en la ventana, el cual le mostrara una lista de los meses, seleccione el mes al cual quiere desplazarse.
  • Día: seleccione un día de los mostrados en el recuadro del calendario de la ventana.

para confirmar la selección pulse sobre el botón seleccionar.

 

volver al principio.
