Creación de un nuevo jefe

La pestaña "admin: Nuevo Jefe"  le permitirá configurar un nuevo jefe para insertarlo en la Base de datos. Para ello deberá rellenar todos los campos que se le muestran en esa pestaña.

Para confirmar la creación del empleado pulse el botón "aceptar", en caso contrario pulse "cancelar" para borrar todos los datos configurados. En caso de no haber rellenado todos los campos de la pestaña o de haberlo hecho de forma incorrecta, al pulsar el botón "aceptar" aparecerá una ventana indicando que campo esta aún sin rellenar o que errores tiene.

A continuación se detallará el funcionamiento y restricciones de los diferentes elementos.

En la pantalla podrá diferenciar dos recuadros para configurar dos tipos de datos

Vendedor:

Se corresponde con el número de vendedor asignado al usuario, este número deberá ser una cifra de 8 dígitos y será el número con el cual el usuario accederá a la aplicación para loguearse.

 

 

volver al principio.

Clave:

Este campo corresponde a la clave personal de un jefe, la cual utilizara junto con su número de vendedor para acceder tanto a la aplicación como a la página web. Esta clave no podrá tener una longitud mayor de 10 caracteres, estos caracteres podrán ser tanto letras como números .

para rellenar este campo se admiten dos opciones:

  • Generación automática: la aplicación le sugerirá una clave de 8 caracteres
  • Generación manual: la aplicación le permitirá introducir por teclado la clave que usted desee.

 

 

volver al principio.

Nombre, 1er apellido, 2º Apellido :

Estos campos se corresponderán con el nombre, 1er apellido, 2º Apellido del usuario que estamos introduciendo, estos campos permiten una longitud de 20 caracteres cada uno de ellos.

 

volver al principio.
