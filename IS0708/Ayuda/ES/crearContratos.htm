Crear un nuevo contrato

Al pulsar el botón "nuevo contrato", se despliega una nueva ventana que le permitirá crear un nuevo contrato

En esta ventana podrá diferenciar tres recuadros:

Contrato: en donde podrá introducir el nombre del contrato nuevo

Turnos: aparecerá una lista de los turnos que vaya añadiendo al contrato actual que está definiendo. A la derecha de este recuadro hay una serie de botones que le permitirá configurar los turnos:

  • Nuevo Turno:
  • Modificar turno:
  • Eliminar turno:
  • Elegir turno:
  • Turno Inicial:

Ciclo:

  • Longitud del ciclo:
  • Configurar:
  • Ayuda:

 

En la parte de abajo de la ventana se encuentran un par de campos de texto en los que podrá configurar el salario y el tipo de contrato.

Una vez configurados todos los parámetros pulse "Aceptar" para confirmar la creación del nuevo contrato, en caso contrario pulse "Cancelar" para descartar el contrato y volver a la pestaña de contratos.

En caso de no haber rellenado todos los datos correctamente aparecerá un aviso para que lo rellene correctamente.

 

Nuevo Turno:

Al pulsar este botón se desplegará una nueva ventana que le permitirá crear un nuevo turno para el contrato actual que esta desarrollando.

Esta ventana contiene dos recuadros:

  • Nombre: en el qué deberá introducir una breve descripción del turno
  • Turno: que le permitirá configurar el horario del turno, para ello pude alargar las barras verdes arrastrándolas desde los extremos o desplazarlas de lugar pinchando sobre ella y deslizándola sobre la superficie

podrá visualizar el horario en intervalos de 5, 10, 15, 30 minutos o 1 hora, para ello seleccione la opción correspondiente en el recuadro de Turno.

Modificar Turno:

Al pulsar este botón se desplegará una nueva ventana que le permitirá modificar un turno seleccionado previamente en la ventana de nuevo contrato, en caso de no seleccionar ningún turno le saldrá un mensaje indicándole que debe seleccionar un turno para acceder a esta opción.

Esta ventana contiene dos recuadros:

  • Nombre: que le permitirá modificar el nombre de este contrato
  • Turno: que le permitirá modificar el horario del turno, para ello pude alargar las barras verdes arrastrándolas desde los extremos o desplazarlas de lugar pinchando sobre ella y deslizándola sobre la superficie

podrá visualizar el horario en intervalos de 5, 10, 15, 30 minutos o 1 hora, para ello seleccione la opción correspondiente en el recuadro de Turno.

 

Volver al principio.

Eliminar Turno:

Al pulsar este botón le permitirá eliminar un turno seleccionado en la ventana de nuevo contrato mostrándole una ventana de confirmación, en caso de no seleccionar ningún turno le saldrá un mensaje indicándole que debe seleccionar un turno para poder eliminarlo.

Una vez eliminado no podrá volver a disponer de este turno, a menos que lo vuelva a crear.

Turno Inicial:

Al pulsar este botón le permitirá marcar como turno inicial al turno seleccionado en la ventana de nuevo contrato,

El turno inicial es el turno con el cual empezará a trabajar el empleado al que se le asigne el contrato que está creando

 

Volver al principio.

Longitud del ciclo:

Le permite indicar el número de días del ciclo del que constará el contrato que está creando. para configurarlo haga "click" con el ratón sobre el botón configurar que se encuentra situado a la derecha.

El ciclo se repite de forma periódica durante la duración del contrato.

 

Volver al principio.

Configurar:

Al pulsar el botón configurar, le permitirá configurar los días del ciclo que indico previamente en el campo longitud de ciclo en la ventana de nuevo contrato.
