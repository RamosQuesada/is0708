Contratos

La pestaña contratos le permitirá visualizar todos los contratos asignados a miembros de su departamento en la tabla que aparece en esta pestaña

En esta pestaña podrá realizar las siguientes opciones:

Crear un nuevo contrato: al seleccionar esta opción se mostrara una ventana para crear un nuevo contrato y configurarle. Para mas información acerca de esta ventana pulse aquí.

Modificar un contrato ya existente: al seleccionar esta opción se mostrara una ventana con los datos actuales del contrato seleccionado para reconfigurarle. Para mas información acerca de esta ventana pulse aquí.

Eliminar un contrato: al seleccionar esta opción se mostrara una ventana de confirmación para verificar si realmente desea borrar el contrato seleccionado.

 

Para seleccionar un contrato haga "click" sobre él con el ratón.
